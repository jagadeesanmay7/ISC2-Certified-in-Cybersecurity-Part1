PINs, passwords, and passphrases are all which type of authentication?

Explanation

Which port is used by our Dynamic Host Configuration Protocol (DHCP) servers to communicate with the clients?

Explanation

OBJ-4.1: DHCP uses UDP Port 67 for the DHCP Server and UDP Port 68 for the Client, it is important to note, those are the destination ports. For this question a DHCP reply from the DHCP server to the client would be from the server with a source port of 67 and a destination port of 68. The initial DHCP request would come from the client with a source port of 68 and a destination port of 67.
